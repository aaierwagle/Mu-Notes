## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/aaierwagle/2025Codes.git
   ```
2. Navigate to the project directory:
   ```bash
   cd 2025Codes
   ```
3. Install dependencies (if applicable):
   ```bash
   npm install
   ```
4. Start the project:
   ```bash
   npm start
   ```
